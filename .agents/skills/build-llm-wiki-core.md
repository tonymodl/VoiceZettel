# Рефакторинг хранилища под LLM-Wiki Pattern

## Директивы кодогенерации

1. **Конфигурация агента**: Напиши скрипт `scripts/setup-openclaw.sh`. Скрипт должен устанавливать Ollama и генерировать конфигурацию `~/.openclaw/openclaw.json`.

2. **Активация прав**: В конфигурации агента установи параметр `"tools": { "profile": "full" }`. Укажи строгий sandbox: агент имеет право читать/писать только в папках `Vault/Raw_v2` и `Vault/Wiki_v2`.

3. **Правила (USER.md)**: Создай профиль агента. Пропиши жесткий паттерн Андрея Карпаты:
   - **Ingest**: чтение файлов из `/Raw_v2`, извлечение фактов, обновление Markdown-файлов в `/Wiki_v2`.
   - Агенту **СТРОГО ЗАПРЕЩЕНО** изменять исходники в `/Raw_v2`.
