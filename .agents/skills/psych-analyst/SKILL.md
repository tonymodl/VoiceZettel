# Psych Analyst — Психологический Анализ и CRM Нового Поколения

## Назначение
Непрерывная фоновая обработка коммуникаций для экстракции 
психологических метаданных и управления приоритетами контактов.

## Мультимерные Метрики

### Emotional Intensity (0.0–1.0)
Численная оценка эмоционального заряда:
- `0.0–0.3` — нейтральный/деловой тон
- `0.3–0.6` — умеренная эмоциональность
- `0.6–0.8` — повышенная (радость/тревога)
- `0.8–1.0` — критическая (конфликт/эйфория)

### Sentiment Trend
Отслеживание изменения тональности:
- Внутри одного диалога (мгновенный тренд)
- На протяжении недель/месяцев (долгосрочный тренд)
- Формат: `{ current: 0.6, 7d_avg: 0.4, 30d_avg: 0.5, trend: "improving" }`

### Financial Commitments
Систематический подход к выявлению денежных обещаний:
```json
{
    "person": "Иванов",
    "type": "receivable",
    "amount": 500000,
    "currency": "RUB",
    "deadline": "2026-06-01",
    "status": "pending",
    "confidence": 0.9,
    "source": "/Raw_v2/Telegram/ivanov_2026-04-15.md",
    "context": "Пообещал оплатить после подписания договора"
}
```

### Trust Score (0.0–1.0)
Оценка надёжности собеседника:
```
trust = (promises_kept / total_promises) * 0.5 +
        response_time_consistency * 0.2 +
        sentiment_stability * 0.15 +
        financial_reliability * 0.15
```

## Круги Данбара

### Автоматическое ранжирование
| Круг | Размер | Критерии |
|------|--------|----------|
| **1** (ближний) | 5 чел | Ежедневное общение, эмоциональная поддержка |
| **2** (друзья) | 15 чел | Еженедельное общение, доверие |
| **3** (знакомые) | 50 чел | Регулярный контакт, рабочие отношения |
| **4** (периферия) | 150 чел | Редкий контакт, формальные отношения |

### Health Score Engine
```
health_score = 
    engagement_index * 0.35 +      # Частота + инициативность
    sentiment_average * 0.25 +     # Средний сентимент за 30d
    reciprocity * 0.20 +           # Баланс диалога (кто кому чаще пишет)
    decay_penalty * 0.20           # Штраф за давность (exp decay по кругу)
```

### Decay Rate по кругам
- Круг 1: `decay_half_life = 7 дней` (быстро падает без контакта)
- Круг 2: `decay_half_life = 30 дней`
- Круг 3: `decay_half_life = 90 дней`
- Круг 4: `decay_half_life = 365 дней`

## Проактивные Триггеры

### Автоматические напоминания
Если `health_score < threshold[dunbar_layer]`:
1. Генерировать черновик сообщения на базе эмоционального контекста
2. Сохранять в `sqlite_v2.db` → таблица `draft_actions`
3. Показывать в Daily Offload Dashboard (колокольчик)

### Календарь обещаний
- Автономно вести скрытый Google Calendar «VZ_Promises»
- Интегрировать дни рождения, дедлайны обещаний, значимые события
- Push-уведомления за 1 день и в день события
